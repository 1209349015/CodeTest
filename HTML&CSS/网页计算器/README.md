# myCalc
一个简易计算器

1. 纯HTML、CSS、JS编写
2. 该计算器布局使用Flex布局
3. 移动APP使用HBuild构建
4. 在APP上，当输入手机号码后长按等于号可以拨打手机号码
5. 作者：dunizb，www.mybry.com

※Build 1240. Version：4.0
